# 🌿 EcoMonitor - Backend API

Bem-vindo(a) ao repositório do backend do **EcoMonitor**! 
Esta é uma API RESTful moderna e rápida, desenvolvida como parte do meu Trabalho de Conclusão de Curso (TCC). Ela é responsável por gerenciar toda a regra de negócio, autenticação de usuários e comunicação com o banco de dados da aplicação.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as ferramentas mais modernas do ecossistema Python:

* **[Python 3](https://www.python.org/):** Linguagem de programação principal.
* **[FastAPI](https://fastapi.tiangolo.com/):** Framework web super rápido para construção da API.
* **[PostgreSQL (Neon.tech)](https://neon.tech/):** Banco de dados relacional Serverless na nuvem.
* **[SQLAlchemy](https://www.sqlalchemy.org/):** ORM (Object Relational Mapper) para manipulação do banco de dados.
* **[Uvicorn](https://www.uvicorn.org/):** Servidor web de alta performance (ASGI) para rodar a aplicação.
* **[Passlib & Bcrypt](https://passlib.readthedocs.io/):** Para criptografia segura das senhas dos usuários.
